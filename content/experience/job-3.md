## Senior Software Engineer (DevOps) at AdHoc LLC

During my time as a Senior Software Engineer (DevOps) at AdHoc LLC, I played a pivotal role in enhancing the DevOps practices for our customers as well as gaining full stack development experience and experience architecting cloud native distributed systems. It was also the first time that I was placed in a position where I was able to lead a team of engineers and help them grow in their careers.

Initially I worked on a project at the US Department of Health and Human Services (HHS) where I set up CI/CD pipelines using Circle CI, containerized applications using Docker and deployed them to Cloud.gov (a PaaS based on Cloud Foundry). I also helped out with full stack development using React and Node.js, which allowed me to gain a deeper understanding of the software development lifecycle and how DevOps practices can be integrated into it.

Most of my time in this role was spent working on software for the US Library of Congress (LOC). I led a team which was asked to modernize the way that the LOC ingests web archives. This was a cloud native distributed system built on AWS using a service oriented architecture. We used Python with AWS Lambda for serverless computing, Amazon S3 for storage, SQS for an event queue, and Amazon DynamoDB for a NoSQL database. This project allowed me to apply my knowledge of cloud-native architectures and distributed systems while also mentoring junior engineers on the team.

Additionally, we implemented infrastructure as code (IaC) using Terraform, and CI/CD pipelines using Gitlab CI/CD. Adding automated testing, security scans and compliance checks to our pipelines was a key focus, ensuring that our deployments were not only efficient but also secure and compliant with government standards.
